fracture by fractureio

A high-performance socket I/O message pipeline library.

Fracture is a high-performance socket I/O library that uses a message pipeline for processing messages. Fracture uses an agent model for isolating state and managing concurrency. It also provides default TCP and HTTP server implementations.

Dependencies

F# 2.0
